# HTTP response codes

## What are they?
- Indicate whether a specific HTTP request has been successfully completed.
- Responses can be categorized into 5 classes:
  - 1xx: Informational
  - 2xx: Successful
  - 3xx: Redirection
  - 4xx: Client error
  - 5xx: Server error

## Common response codes
- 200: OK
- 201: Created
- 301: Moved permanently
- 400: Bad request
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 500: Internal server error
